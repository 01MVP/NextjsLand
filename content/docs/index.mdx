Welcome to NextjsLand Template! This is a multilingual documentation website template built with Fumadocs and Next.js.

## What is Fumadocs?

Fumadocs is a modern documentation framework designed for building beautiful, fast, and maintainable documentation websites. It's built on Next.js and provides rich features and components.

## Template Features

- 🌍 **Multilingual Support** - Supports Simplified Chinese, Traditional Chinese, and English
- 🎨 **Modern Design** - Beautiful interface based on Tailwind CSS
- 🚀 **High Performance** - Server-side rendering with Next.js
- 📱 **Responsive Design** - Perfect adaptation to various devices
- 🔍 **Full-text Search** - Built-in powerful search functionality
- 📝 **MDX Support** - Support for React components in Markdown
