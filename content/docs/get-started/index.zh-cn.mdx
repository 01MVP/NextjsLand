欢迎使用 NextjsLand模板！本指南将帮助您快速上手这个强大的多语言文档模板。

## 概述

这个模板基于 Fumadocs 和 Next.js 构建，提供了完整的多语言文档网站解决方案。它包含了现代化的设计、强大的功能和易于使用的组件系统。

## 主要特性

- **多语言支持**: 内置中文简体、中文繁体和英文支持
- **现代化框架**: 基于 Next.js 15 和 React 19
- **优秀的开发体验**: TypeScript 支持，热重载，快速构建
- **SEO 友好**: 服务端渲染，自动生成 sitemap
- **响应式设计**: 完美适配桌面端和移动端
