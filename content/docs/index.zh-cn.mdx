欢迎使用 NextjsLand模板！这是一个基于 Fumadocs 和 Next.js 构建的多语言文档网站模板。

## 什么是 Fumadocs？

Fumadocs 是一个现代化的文档框架，专为构建美观、快速且易于维护的文档网站而设计。它基于 Next.js 构建，提供了丰富的功能和组件。

## 模板特性

- 🌍 **多语言支持** - 支持中文简体、中文繁体和英文
- 🎨 **现代化设计** - 基于 Tailwind CSS 的美观界面
- 🚀 **高性能** - 基于 Next.js 的服务端渲染
- 📱 **响应式设计** - 完美适配各种设备
- 🔍 **全文搜索** - 内置强大的搜索功能
- 📝 **MDX 支持** - 支持在 Markdown 中使用 React 组件
